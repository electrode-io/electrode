AssetPathMap

AssetPathMap: { base: string } & Record<string, string>
  • Defined in node/types.ts:39

mapping of asset path by file extensions

  • extensions must have . prefix: .js
  • base must be provided

ie:

{
  base: "/assets",
  ".js": "/js",
  ".css": "/css"
}
