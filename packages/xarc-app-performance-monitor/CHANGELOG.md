# Changelog

## 1.0.0

### Features

- Initial release
- Web Vitals tracking (LCP, FID, FCP)
- Bundle load time and size metrics
- Performance timing metrics (DNS, TCP, TTFB, etc.)
- Error tracking
- Auto-initialization for zero-config usage
- Manual API for custom integration
- SSR-safe implementation
- TypeScript support with full type definitions
- ESM and CJS builds
